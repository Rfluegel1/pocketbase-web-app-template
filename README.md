# web-app-template
This is a web application template making use of pocketbase, node, and react

## First Time Setup

1. ### Install Dependencies
```npm install```

2. ### Run pocketbase
1. ```./pocketbase serve```
2. Create your admin account at the admin UI ( http://127.0.0.1:8090/_/ by default)

3. ### Run tests
```npm run test```
